## Plugins

### [MediathekViewWeb - Download Plugin](https://github.com/CatNoir2006/jellyfin-plugin-MediathekViewDL)
Mit diesem Plugin kannst du einfach Inhalte der öffentlich-rechtlichen Mediatheken über Jellyfin suchen und herunterladen. Für die Suche wird die API von MediathekViewWeb genutzt. So kannst du Sendungen und Filme der öffentlich-rechtlichen Sender komfortabel direkt in deiner Jellyfin-Umgebung verfügbar machen und verwalten. Weitere Informationen und Installationshinweise findest du im [Original-Repository](https://github.com/CatNoir2006/jellyfin-plugin-MediathekViewDL).
